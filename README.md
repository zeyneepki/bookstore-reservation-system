# Bookstore Reservation & Management System

## Overview

This project is a Java Swing-based desktop bookstore automation system integrated with MySQL using JDBC.

The application provides a complete user authentication system, book browsing and searching features, reservation functionality, and an administrative panel for managing book records.

It was developed to demonstrate database integration, GUI design, and CRUD operations within a desktop environment.

## Features

- User Registration with input validation
- User Login Authentication
- Admin Login Panel
- Book Listing and Viewing
- Book Search Functionality
- Book Reservation System
- Admin Book Management Add / Delete / Update
- MySQL Database Integration via JDBC
- Form validation using regular expressions

## Technologies Used

- Java OOP Principles
- Java Swing GUI Development
- MySQL
- JDBC Database Connectivity
- NetBeans IDE

## System Architecture

The system consists of:

- Role-based access User / Admin
- Database-driven structure
- Multi-form GUI design
- Secure authentication logic
- Structured CRUD operations for data management

## Purpose of the Project

The main objective of this project is to:

- Apply object-oriented programming concepts
- Implement database connectivity in a desktop application
- Design a functional bookstore management system
- Practice authentication and data validation mechanisms
